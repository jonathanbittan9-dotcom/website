# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Overview

This is a personal learning workspace for web development and programming. There is no build system, package manager, or test suite — files are opened directly in a browser or run with the appropriate runtime.

## Running files

- **HTML/CSS files**: Open directly in a browser (no server needed)
- **Python files**: `python pythonhomework/exeN.py`
- **Java files**: `javac physics.java && java physics`

## Repository structure

| Directory | Purpose |
|---|---|
| `html-learning/` | HTML fundamentals practice (links, forms, tables, images, video) |
| `css-learning/` | CSS fundamentals practice (selectors, fonts, borders, shadows) |
| `html-develop/` | More polished HTML projects |
| `css+html_develop/` | Combined CSS+HTML projects (in progress) |
| `javascript-learning/` | JS fundamentals practice (in progress) |
| `javascript-develop/` | JS projects (in progress) |
| `project-useall/` | Full projects combining HTML, CSS, and JS |
| `pythonhomework/` | Python homework exercises (loops, arithmetic, input/output) |
| `images/` | Shared media assets (images, audio, video) |
